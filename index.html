StudySmart: AP Chemistry

StudySmart allows students to access prior AP Exam questions in a more targeted manner for more effective studying.

1) Unzip the folder once you have downloaded it.
2) Open the "dist" folder and then click on the file titled "main".
This will open the application.

Why so many files?

The other files ensure the application runs on your computer regardless of whether you have python or other relevant libraries installed.


How it works:

I used Natural Language Processing to analyze previously released AP Chemistry questions and trained a model to classify questions into their respective units based on some questions that were already classified from other sources and myself. The following application allows you to access this information. You can also help me fine tune the model by providing feedback for incorrectly classified questions.
